Requirements Specification for Ctrl-Alt-Defeat

1. Introduction

1.1 Purpose of Product

Test users typing speed while challenging them to a fun interactive game. Encourage user to type as fast as possible to avoid being eaten by the monster chasing them. User will progress through various levels and be confronted by different challenges that get increasingly more difficult. Users skill and progress will be tracked through a high score table that shows their fastest typing speed.


1.2 Scope of Product

Game will include a minimum of three levels of increasing difficulty that will increase both speed user is required to type as well as length of the words themselves. In higher difficulty levels user will be required to type specific action words that enable the character to jump over or crouch under obstacles. When character runs into monster chasing them the game will end. Lastly if user types correct words then the speed of the character will increase and incorrect words will make it slow down. Game will have a start menu that also has a high score that maintains the users fastest typing speed.


1.3 Acronyms, Abbreviations, Definitions

playerX/playerY: Controls character movement (Jumping, crouching, speeding up, and slowing down). worldX/worldY: Controls world movement (Makes it appear that character is running through world). monsterX/monsterY: Sets monsters position (Monster does not move, only animation of monster changes). spriteCounter: Controls speed in which animation of sprite changes (Expresses speed of sprites movement).
